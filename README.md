# INSTALACAO

Abra o terminal no diretorio da aplicacao e instale as dependências com o comando:

``npm install``

Apos a instalacao de dependências voce pode inicializar o projeto com o comando:

``ng serve --open``

# CREDITOS

- Para esse projeto, nenhum template foi utilizado. O tema e layout foram criados do zero.

- Foi utilizado a versao mais recente do Bootstrap - https://getbootstrap.com/
